# # 倒计时应用

这是一个简单易用的网页版倒计时器，可以帮助您追踪重要事件的到来。

## 功能特性

- 直观的倒计时显示（天、小时、分钟、秒）
- 可自定义目标日期和时间
- 支持为事件命名
- 数据持久化（使用浏览器本地存储）
- 响应式设计，支持移动设备
- 美观的渐变背景和卡片式界面

## 使用方法

1. 打开页面后，设置您想要的目标日期和时间
2. 输入事件名称（可选）
3. 点击“设置倒计时”按钮开始倒计时
4. 随时可以点击“重置”按钮重新开始

## 技术栈
