# 空中自动驾驶辅助降落系统

一个现代化的空中自动驾驶辅助降落界面，支持用户在下视摄像头上点击指定位置进行导航降落。

## 功能特性

- **实时摄像头视图**: 显示下视摄像头的实时画面
- **点击导航降落**: 用户可以直接在摄像头视图上点击选择降落位置
- **实时状态监控**: 显示高度、速度、电池电量等关键飞行参数
- **智能降落模式**: 支持精准降落和紧急降落模式
- **安全检测**: 障碍物和地面类型检测
- **交互式控制**: 完整的飞行控制面板

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **图形渲染**: Canvas API
- **UI设计**: 现代化扁平设计，响应式布局

## 界面组件

### 1. 摄像头视图区域
- 实时摄像头画面显示
- 网格覆盖辅助定位
- 降落目标标记
- 当前位置指示器

### 2. 状态栏
- 实时高度显示
- 飞行速度监控
- 电池电量显示
- 飞行状态指示

### 3. 控制面板
- 起飞/降落控制
- 模式切换
- 安全选项配置

### 4. 降落信息面板
- 目标坐标显示
- 预计到达时间
- 下降速率
- 安全等级评估

## 使用方法

1. **起飞**: 点击"起飞"按钮开始飞行
2. **选择降落点**: 在摄像头视图上点击选择降落位置
3. **开始降落**: 点击"自动降落"按钮开始自动降落
4. **取消降落**: 在降落过程中可点击"取消"按钮中断降落
5. **模式切换**: 可在自动模式和手动模式之间切换

## 安全特性

- 防重复点击机制
- 降落过程实时监控
- 安全等级评估
- 紧急停止功能

## 响应式设计

界面适配不同屏幕尺寸，支持桌面端和移动端使用。
